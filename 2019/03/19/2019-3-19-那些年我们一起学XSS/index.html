常规来说，因为 onxxxx="[输出]"和 href="javascript:[输出]" 与 <script>[输出]</script>没有太大区别。因为[输出]所在的地

Tips:

在 HTML 属性中，会自动对实体字符进行转义。一个简单的比方。

<img src="1" onerror="alert(1)">

和

<img src="1" onerror="alert&#x28;1&#x29;">

是等效的

换言之，只要上面的情况，没有过滤 &，# 等符号，我们就可以写入任意字符。
